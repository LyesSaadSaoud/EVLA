## Title

EVLA: Energy Vision-Language-Action Benchmark for Residential Energy Management

## Description

EVLA is a physics-grounded multimodal benchmark for intent-conditioned residential energy management. The release contains dataset metadata, validation summaries, paper figures, reproducibility tables, code snapshots, and selected sample records associated with the EVLA paper.

The benchmark contains 439,203 retained base operating states expanded across three hidden visual-energy regimes and five language objectives, producing 6,588,045 multimodal samples. Each sample combines an RGB energy-field representation, explicit physical state metadata, a natural-language objective, and a 16-step battery-action trajectory generated by a physics-consistent shooting-MPC oracle.
