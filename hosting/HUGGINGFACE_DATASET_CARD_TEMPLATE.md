# EVLA: Energy Vision-Language-Action Benchmark

EVLA is a physics-grounded multimodal benchmark for intent-conditioned residential energy management. Each sample combines:

- RGB energy-field representation
- explicit physical state vector
- natural-language objective
- 16-step battery-action trajectory generated by a physics-consistent shooting-MPC oracle

## Dataset Summary

- Total multimodal samples: 6,588,045
- Base operating states: 439,203
- Hidden visual-energy regimes: 3
- Language objectives: 5
- Counterfactual worlds per base state: 15
- Action horizon: 16 steps

## Sample Fields

```text
image: RGB energy-field representation
state: price, SOC, indoor temperature, hour
language: natural-language objective paraphrase
action: 16-step normalized battery-action trajectory
metadata: base-state id, hidden regime, objective class
```

## Intended Use

EVLA is intended for research on multimodal, intent-conditioned, and physics-grounded learning for residential energy management.

## Not Intended Use

EVLA is not a certified building-control system and should not be used directly as a deployment-ready home-energy controller.
